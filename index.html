Git Commands

Inilize a git repository

git init

Download GitHub repository locally

git clone <URL>

Add new files to the repository

git add <file name>

Save new changes

git commit -m "message"

Check status of the repository

git status

Add new files and commit at the same time

git commit -am "message"

Upload changes to GitHub

git push

Download latest version from GitHub

git pull

Tracl all commits

git log

Retrieve to a previous version

git --hard <commit hash>

git --hard origin/master

Check currnt branch

git branch

Create new branch

git checkout -b <branch name>

Switch to other branch

git checkout <branch name>

Merge branches

git merge <branch name>
